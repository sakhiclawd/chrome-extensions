# SCREENSHOT_GUIDE.md - Briefly Analyzer

To capture high-resolution screenshots for the Chrome Web Store:

### 1. Enable Demo Mode
The extension includes a **Demo Mode** to provide high-signal data for screenshots.
*   Navigate to `https://www.pmdirectory.net`.
*   Open the extension.
*   The title will change to **BRIEFLY ANALYZER (DEMO)** and populate with impressive metrics.

### 2. Capture Specifications
*   **Dimensions:** Use a window size of **1280x800** or larger.
*   **Format:** High-quality PNG.
*   **Browser:** Chrome with a clean profile (no other extensions visible).

### 3. Required Views
| View | Description |
| :--- | :--- |
| **Main Summary** | Capture the `Summary` tab with the "Dominant" status and insights. |
| **Traffic Data** | Switch to the `Traffic` tab to show the "1.2M+ visits" metric. |
| **SEO Visibility** | Capture the `Keywords` tab showing the "$185K" est. cost. |
| **Authority** | Capture the `Backlinks` tab showing the "840K" backlinks. |

### 4. Tips for Quality
*   Keep the mouse cursor away from the popup area.
*   Ensure the "Save Snapshot" button is clearly visible.
*   Crop precisely around the popup border if not using a dedicated tool.
