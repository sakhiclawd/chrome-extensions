# PRIVACY_POLICY.md - Briefly Analyzer

## Overview
This Privacy Policy describes how Briefly Analyzer ("the extension") handles user data. Our primary goal is to provide website metrics and SEO analysis on user request without compromising privacy.

## Data Collection
- **Active Tab Domain:** The extension only reads the domain and hostname of the active tab when the user explicitly clicks the extension icon. This is necessary to fetch metrics for the specific site being analyzed.
- **User-Initiated Reports:** If the user chooses to "Save Snapshot," the domain, its metrics, and the analysis timestamp are sent to `pmdirectory.net` to be stored in the user's PMLibrary.
- **No Browsing History:** We do NOT collect, store, or sell your browsing history, keystrokes, or personal identification.

## Permissions Usage
- **activeTab:** Used only to identify the current website for analysis upon user request.
- **scripting:** Used to perform domain extraction from the active webpage.
- **storage:** Used to store local preferences.
- **host_permissions:** Specifically for communicating with `pmdirectory.net` to fetch and save domain metrics.

## Third-Party Services
Metrics are fetched from our backend SEO data proxy. No personally identifiable information (PII) is shared with third-party data providers.

## Changes to This Policy
We may update this policy occasionally. Continued use of the extension constitutes acceptance of any changes.
